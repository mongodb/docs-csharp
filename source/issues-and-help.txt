Issues & Help
=============

.. default-domain:: mongodb

Our developer community is vibrant and highly engaged, with extensive experience using C#/.NET with MongoDB.

Often, the quickest way to get support for general questions is through the
`MongoDB Community Forums <https://www.mongodb.com/community/forums>`__.

Refer to our :manual:`support channels </support>` documentation for more information.

Bugs / Feature Requests
-----------------------

To report a bug or to request a new feature in the C#/.NET driver,
please open a case in our issue management tool, JIRA:

* `Create an account and login <https://jira.mongodb.org>`_.
* Navigate to `the .NET Driver project <https://jira.mongodb.org/browse/CSHARP>`_.
* Click **Create Issue**. Please provide as much information as possible about the
  issue and the steps to reproduce it.

Bug reports in JIRA for the C#/.NET driver and the Core Server (i.e. SERVER) project are **public**.

Pull Requests
-------------

We are happy to accept contributions to help improve the driver. We will guide user contributions to ensure they meet the standards of the codebase. Please ensure that any pull requests include documentation, tests, and pass the existing test suite.

To get started check out the source and work on a branch:

.. code-block:: bash

   $ git clone https://github.com/mongodb/mongo-csharp-driver.git
   $ cd mongo-csharp-driver
   $ git checkout -b myNewFeature

Finally, follow the `Testing/Development guidelines
<https://github.com/mongodb/mongo-csharp-driver/blob/master/CONTRIBUTING.md>`__ to
ensure your code passes any newly added and existing tests.
