Expected Result
~~~~~~~~~~~~~~~

Running any of the preceding full examples prints the following results:

.. code-block:: none

   // results truncated
   { 
      "_id" : ObjectId("5eb3d668b31de5d588f42950"), 
      "name" : "Bagels N Buns", 
      "restaurant_id" : "40363427", 
      "cuisine" : "Delicatessen", 
      "address" : {...}, 
      "borough" : "Staten Island", 
      "grades" : [...] 
   }
